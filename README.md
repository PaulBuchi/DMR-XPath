# DMR-XPath

## Kurzanleitung

- Starte das Hauptprogramm mit:
  ```
  python main.py
  ```

- Wähle die gewünschte Phase im Menü (Toy-Beispiel, DBLP-Verarbeitung, Optimierungen).
- Die wichtigsten Ergebnisse und Statistiken werden im Terminal ausgegeben.
